## Remaining before freezing (see also eval/README.md)

1. Human spot-check ~20% of tier B (cross_document, multi_hop, aggregation) —
   these carry the baseline-vs-agentic conclusion.
2. On Colab (with the real index): run `eval/retrieval_eval.py --verify`
   against golden_set_v2 and repair/drop evidence spans that don't match
   indexed chunk text (spans are verbatim PDF text via Drive extraction;
   docling's extraction can differ in spacing/hyphenation).
3. Freeze golden_set_v2.json — do not tune against it.
4. Run the baseline: `python eval/run_benchmark.py --pipeline baseline`
   (per-question JSONL resume), then score/compare tooling.

## Protocol used (for reproducing / topping up)

Agents follow AGENT_PROMPT_TEMPLATE.md; papers assigned from
papers_manifest.tsv excluding papers_used.txt; each batch file is
independently valid JSON so any interrupted wave loses nothing. Re-run
`python eval/golden_v2_work/assemble.py` after adding batches, then commit
and push (branch `claude/rag-uhtc-v2-folder-tryfnw`).
